◆ initWithCSRElementsTLV

nullable instancetype initWithCSRElementsTLV

Initialize an WTLOperationalCSRInfo by providing just the csrElementsTLV and attestationSignature (which can come from an WTLOperationalCredentialsClusterCSRResponseParams). This will extract the csr and csrNonce from the csrElementsTLV, if possible, and return nil if that fails.

◆ initWithCSRNonce

nullable instancetype initWithCSRNonce

Initialize an WTLOperationalCSRInfo by providing the csrNonce (for example, the nonce the client initially supplied), and the csrElementsTLV and attestationSignature that the server returned. This will ensure that csrNonce matches the data in csrElementsTLV, returning nil if it does not, and extract the csr from csrElementsTLV.
