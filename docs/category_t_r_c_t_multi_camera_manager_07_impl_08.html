success: (RCTResponseSenderBlock)  success

获取设备配置信息，返回一个字典。 查询结果通过字典返回，查询结果 key 与参数里的 key 同名。注意：如果客户端没有查询到参数里的 key 的信息，则结果字典里不会有对应的 key。 例如 params 字典里包含 isSupportedTalk 这个 key，那么回调结果的字典里也会有 isSupportedTalk 这个key，value 是 Boolean 类型，代表设备是否支持对讲。

@available 1.0.0

Parameters
paramsdevId：String，设备id。 query：String，查询项。支持多个，以逗号隔开。例如 isSupportedTalk,isSupportedSound

支持的查询项： isSupportedTalk：返回值类型 Boolean，设备是否支持对讲。在首次打开摄像头面板时，需要在 p2p 通道连接后调用。之后会从缓存中取值。 isSupportedSound：返回值类型 Boolean，设备是否支持拾音器。在首次打开摄像头面板时，需要在 p2p 通道连接后调用。之后会从缓存中取值。 supportedAudioMode：返回值类型 Integer，设备支持的对讲模式。0 未知；1：单向对讲；2：双向对讲。 isSupportedCloudStorage：返回值类型 Boolean，设备是否支持云存储功能。 obtainCameraConfigInfo：旧插件同名方法，获取设备 config 信息，结果字典里包含多个 key，不包含 password, p2pConfig, p2pId 这三个字段
