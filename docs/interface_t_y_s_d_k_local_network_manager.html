◆ requestAuthorization [1/4]

- (void) requestAuthorization

Requests permissions on the local network (beta)

If you call this method when a Wi-Fi network is connected, a process is started to request permissions on the local network from the user. The authorization prompt will appear only once until the app is uninstalled. Currently, you cannot get an authorization change callback for this request.
