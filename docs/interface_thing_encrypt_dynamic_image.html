◆ preloadAllAnimatedImageFrames

- (BOOL) preloadAllAnimatedImageFrames
readwritenonatomicassign

Preload all frame image to memory.

@discussion Set this property to YES will block the calling thread to decode all animation frame image to memory, set to NO will release the preloaded frames. If the image is shared by lots of image views (such as emoticon), preload all frames will reduce the CPU cost.

See animatedImageMemorySize for memory cost.
