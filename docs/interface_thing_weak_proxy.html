Detailed Description

A proxy used to hold a weak object. It can be used to avoid retain cycles, such as the target in NSTimer or CADisplayLink.

sample code:

@implementation MyView { NSTimer *_timer; }

  • (void)initTimer { ThingWeakProxy *proxy = [ThingWeakProxy proxyWithTarget:self]; _timer = [NSTimer timerWithTimeInterval:0.1 target:proxy selector:@selector(tick:) userInfo:nil repeats:YES]; }
  • (void)tick:(NSTimer *)timer {...} @end
