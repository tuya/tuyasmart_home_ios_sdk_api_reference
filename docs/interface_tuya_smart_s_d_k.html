Detailed Description

TuyaSmartSDK is an entry to use the Tuya SDK.

Before you use the SDK, go to the Tuya IoT Platform (https://iot.tuya.com) to create an SDK app. The following data from the platform is required to initialize the SDK:

  • Tuya App Key.
  • Tuya App Secret.
  • Security image: renamed to t_s.bmp and put in your project as a resource.
  • Bundle ID: your app's bundle ID.

To set parameters in this class, such as appGroupId, env, appVersion, and lang, set them before you initialize the SDK.

Then, call this method to initialize the SDK: [sharedInstance (TuyaSmartSDK) startWithAppKey:@"YOUR_APP_KEY" secretKey:@"YOUR_APP_SECRET"];.
