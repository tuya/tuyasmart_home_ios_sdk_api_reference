This macro ensures that key path exists at compile time. Given a receiver type and a key path, it verifies at compile time that the key path exists, without calling it.

For example:

ThingKVOClassKeyPath(NSString, length) => "length" ThingKVOClassKeyPath(NSString, lowercaseString.length) => @"lowercaseString.length"

This macro ensures that key path exists at compile time. Given a real receiver with a key path as you would call it, it verifies at compile time that the key path exists, without calling it.

For example:

ThingKVOKeyPath(string.length) => "length" Or even the complex case: ThingKVOKeyPath(string.lowercaseString.length) => @"lowercaseString.length".
