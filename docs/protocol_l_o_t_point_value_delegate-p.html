LOTPointValueDelegate is called at runtime to override the point value of a property in a LOTAnimation. The property is defined by at LOTKeypath. The delegate is set via setValueDelegate:forKeypath on LOTAnimationView. @discussion LOTValueDelegates are used to dynamically change animation data at runtime. A delegate is set for a keypath, defined by LOTKeypath. While the animation is running the delegate is asked for the value for the keypath at each frame of the animation. The delegate is given the computed animation value for the the current frame. See LOTKeypath and the setValueDelegate:forKeypath method on LOTAnimationView.

Parameters
currentFrameThe current frame of the animation in the parent compositions time space.
startKeyframeWhen the block is called, startFrame is the most recent keyframe for the keypath in relation to the current time.
endKeyframeWhen the block is called, endFrame is the next keyframe for the keypath in relation to the current time.
interpolatedProgressA value from 0-1 that represents the current progress between keyframes. It respects the keyframes current easing curves.
startPointThe point from the previous keyframe in relation to the current time.
endPointThe point from the next keyframe in relation to the current time.
interpolatedPointThe point interpolated at the current time between startPoint and endPoint. This represents the keypaths current point for the current time.
Returns
CGPoint the point to set the keypath node for the current frame
