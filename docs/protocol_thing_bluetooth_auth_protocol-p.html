- (ThingCBAuthorizationState) systemPowerStatus

获取系统蓝牙开关状态

此状态不稳定，只存储了一份系统状态， 准确的状态获取应该通过 addObserver:bleStateChanged: 方法

注意：当 APP 蓝牙权限状态为 AllowedAlways 时，此状态才可能返回 PoweredOff 或 PoweredOn; APP 蓝牙权限状态为 NotDetermined 或 Denied 时，此状态返回 Unauthorized； 注意使用 单例 调用
