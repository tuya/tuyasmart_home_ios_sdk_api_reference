Parameters
languageKey多语言key
isFound是否找到了多语言翻译
isCurrent当isFound为True时，这个字段才有效。是否在main bundle(不是默认的语言bundle)中取到的多语言翻译。
localizedString多语言翻译的内容，如果isFound为NO，则localizedString==languageKey
