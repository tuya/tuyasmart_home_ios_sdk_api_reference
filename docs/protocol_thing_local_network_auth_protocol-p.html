◆ requestLocalNetworkAuthorization [1/4]

+ (void) requestLocalNetworkAuthorization

根据来源请求本地网络权限（触发系统弹窗）

申请“本地网络”授权（beta）

在 Wi-Fi 已连接的前提下，调用此方法会触发“本地网络”的授权弹窗。弹窗只会出现一次，直到 App 卸载重装。 目前没有办法获取此次触发的状态变更回调。

Request "Local Network" authorization (beta)

When Wi-Fi is connected, calling this method will start the process of requesting "Local Network" authorization from the user. The authorization prompt will only appear once until App uninstalled. Currently we can't get authorization change callback for this request.
