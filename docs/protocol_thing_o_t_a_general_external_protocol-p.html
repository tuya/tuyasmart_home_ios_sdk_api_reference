- (BOOL) didTapCancelForceUpgrade
optional

取消强制升级事件自定义处理

  1. 强制升级情况，检测升级中，升级弹框点击“取消” 按钮事件。
  2. 强制升级情况，点击 navigationBar 返回按钮（有进度情况会有弹框，点击“确定”按钮）。
Returns
YES 自定义处理，NO 默认 [self.navigationController popToRootViewControllerAnimated:YES]
