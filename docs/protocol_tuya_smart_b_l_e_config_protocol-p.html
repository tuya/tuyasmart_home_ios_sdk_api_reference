Sends an OTA package to update the firmware.

The values of otaData, otaType, and otaVersion can be obtained from TuyaSmartFirmwareUpgradeModel. You can get TuyaSmartFirmwareUpgradeModel by using getFirmwareUpgradeInfo:failure: (TuyaSmartDevice).

Note
Your device must be connected over Bluetooth before the update.
