Detailed Description

The protocol definition for the WTLOTAProviderDelegate

All delegate methods will be called on the supplied Delegate Queue.

While the selectors on this protocol are marked @optional, in practice an implementation must provide an implementation for one of each pair of selectors (e.g. one of the two handleQueryImageForNodeID selectors must be implemented). The selector ending in "completion:" will be used if present; otherwise the one ending in "completionHandler:" will be used.
