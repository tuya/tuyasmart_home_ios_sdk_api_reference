\doxysubsection{Detailed Description}
Category that adds built-\/in {\ttfamily KVOController} and {\ttfamily KVOController\+Non\+Retaining} on any instance of {\ttfamily \mbox{\hyperlink{class_n_s_object-p}{NSObject}}}.

This makes it convenient to simply create and forget a {\ttfamily \mbox{\hyperlink{interface_thing_k_v_o_controller}{Thing\+KVOController}}}, and when this object gets dealloc\textquotesingle{}d, so will the associated controller and the observation info.
