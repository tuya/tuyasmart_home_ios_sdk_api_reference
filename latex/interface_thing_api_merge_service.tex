\doxysubsection{Detailed Description}
\mbox{\hyperlink{interface_thing_api_merge_service}{Thing\+Api\+Merge\+Service}} is used for making several requests in one HTTP request. 

This class is used for reduce the number of network requests. Use it with the following steps\+:
\begin{DoxyEnumerate}
\item Create a \mbox{\hyperlink{interface_thing_api_merge_service}{Thing\+Api\+Merge\+Service}} instance
\item Add several request by {\ttfamily -\/\mbox{[}\mbox{\hyperlink{interface_thing_api_merge_service}{Thing\+Api\+Merge\+Service}} add\+Api\+Request\+:post\+Data\+:version\+:\mbox{]}}
\item Send\+Requests by {\ttfamily -\/ \mbox{[}\mbox{\hyperlink{interface_thing_api_merge_service}{Thing\+Api\+Merge\+Service}} send\+Request\+:success\+:failure\mbox{]}}. requests will be sent and received together.
\end{DoxyEnumerate}

@discussion Due to HTTP/2 multiplexing technology, it is not recommend for use now.
