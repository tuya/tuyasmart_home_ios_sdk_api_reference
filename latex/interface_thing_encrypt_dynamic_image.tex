{\footnotesize\ttfamily -\/ (BOOL) preload\+All\+Animated\+Image\+Frames\hspace{0.3cm}{\ttfamily [read]}, {\ttfamily [write]}, {\ttfamily [nonatomic]}, {\ttfamily [assign]}}

Preload all frame image to memory.

@discussion Set this property to {\ttfamily YES} will block the calling thread to decode all animation frame image to memory, set to {\ttfamily NO} will release the preloaded frames. If the image is shared by lots of image views (such as emoticon), preload all frames will reduce the CPU cost.

See {\ttfamily animated\+Image\+Memory\+Size} for memory cost.
