\doxysubsection{Detailed Description}
An image encoder to encode image to data.

@discussion It supports encoding single frame image with the type defined in Thing\+Image\+Type. It also supports encoding multi-\/frame image with GIF, APNG and WebP.

Example\+:

Thing\+Image\+Encoder $\ast$jpeg\+Encoder = \mbox{[}\mbox{[}Thing\+Image\+Encoder alloc\mbox{]} init\+With\+Type\+:Thing\+Image\+Type\+JPEG\mbox{]}; jpeg\+Encoder.\+quality = 0.\+9; \mbox{[}jpeg\+Encoder add\+Image\+:image duration\+:0\mbox{]}; NSData jpeg\+Data = \mbox{[}jpeg\+Encoder encode\mbox{]};

Thing\+Image\+Encoder $\ast$gif\+Encoder = \mbox{[}\mbox{[}Thing\+Image\+Encoder alloc\mbox{]} init\+With\+Type\+:Thing\+Image\+Type\+GIF\mbox{]}; gif\+Encoder.\+loop\+Count = 5; \mbox{[}gif\+Encoder add\+Image\+:image0 duration\+:0.\+1\mbox{]}; \mbox{[}gif\+Encoder add\+Image\+:image1 duration\+:0.\+15\mbox{]}; \mbox{[}gif\+Encoder add\+Image\+:image2 duration\+:0.\+2\mbox{]}; NSData gif\+Data = \mbox{[}gif\+Encoder encode\mbox{]};

\begin{DoxyWarning}{Warning}
It just pack the images together when encoding multi-\/frame image. If you want to reduce the image file size, try imagemagick/ffmpeg for GIF and WebP, and apngasm for APNG. 
\end{DoxyWarning}
