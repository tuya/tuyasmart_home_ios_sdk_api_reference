\doxysubsection{Detailed Description}
Thing\+KVStorage is a key-\/value storage based on sqlite and file system. Thingpically, you should not use this class directly.

@discussion The designated initializer for Thing\+KVStorage is {\ttfamily init\+With\+Path\+:}. After initialized, a directory is created based on the {\ttfamily path} to hold key-\/value data. Once initialized you should not read or write this directory without the instance.

\begin{DoxyWarning}{Warning}
The instance of this class is {\itshape NOT} thread safe, you need to make sure that there\textquotesingle{}s only one thread to access the instance at the same time. If you really need to process large amounts of data in multi-\/thread, you should split the data to multiple KVStorage instance (sharding). 
\end{DoxyWarning}
