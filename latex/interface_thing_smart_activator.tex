Starting with i\+OS 12, calls to this function will return nil by default, and will only return the correct value if \char`\"{}\+Access Wi\+Fi Information\char`\"{} is enabled in the Xcode project. This function needs to be activated in the App IDs on the developer page in order to use it. Starting with i\+OS 13, at least one of the following three conditions must also be met.
\begin{DoxyItemize}
\item An app that has been granted Location Services permissions.
\item A VPN application that is currently enabled.
\item Use of NEHotspot\+Configuration (only Wi-\/\+Fi networks configured through the app are supported).
\end{DoxyItemize}
