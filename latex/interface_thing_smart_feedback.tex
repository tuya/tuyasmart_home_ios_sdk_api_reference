\doxysubsection{Detailed Description}
\mbox{\hyperlink{interface_thing_smart_feedback}{Thing\+Smart\+Feedback}} provides methods for developers to implement feedback functionalities. 

When the user finds a problem and needs to submit feedback, he/she can add it. When adding feedback, he/she should first select the feedback type, then write the feedback content and submit it.

All functions related to feedback correspond to the \mbox{\hyperlink{interface_thing_smart_feedback}{Thing\+Smart\+Feedback}} class, which supports the following list of functions\+:
