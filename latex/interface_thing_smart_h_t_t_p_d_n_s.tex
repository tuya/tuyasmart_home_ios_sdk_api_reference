\doxysubsection{Detailed Description}
\mbox{\hyperlink{interface_thing_smart_h_t_t_p_d_n_s}{Thing\+Smart\+HTTPDNS}} is another solution for DNS query instead of system DNS service. 

The DNS query can be done through http to prevent DNS cache pollution.
