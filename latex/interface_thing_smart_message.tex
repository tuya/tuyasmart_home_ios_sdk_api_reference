\doxysubsection{Detailed Description}
\mbox{\hyperlink{interface_thing_smart_message}{Thing\+Smart\+Message}} provides methods to implement message functionalities. 

\mbox{\hyperlink{interface_thing_smart_message}{Thing\+Smart\+Message}} provides a message center and supports push functions. The messages are classified into the following categories\+: alert messages, home messages, and notifications. Each type of message can be enabled or disabled.
