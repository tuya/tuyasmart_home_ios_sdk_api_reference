\doxysubsection{Detailed Description}
\mbox{\hyperlink{interface_thing_smart_s_d_k}{Thing\+Smart\+SDK}} is an entry for using the Thing SDK. 

Before using the SDK, please go to Thing IoT Platform (\href{https://iot.thing.com}{\texttt{ https\+://iot.\+thing.\+com}}) create a SDK App. We need 4 things from the Platform to initialize the SDK\+:
\begin{DoxyItemize}
\item Thing App Key.
\item Thing App Secret.
\item Security Image. Rename to {\ttfamily t\+\_\+s.\+bmp} and put it in your project as a resource.
\item Bundle ID. Must equal to your App\textquotesingle{}s bundle ID.
\end{DoxyItemize}

If you need to set some params (app\+Group\+Id, env, app\+Version, lang, etc...) in this class, be sure to set them before initialize the SDK.

Finally, call this method to initialize the SDK\+: {\ttfamily \mbox{[}\mbox{\hyperlink{interface_thing_smart_s_d_k_a4ebb340876b69c9f4b9b4ed4e6232786}{shared\+Instance (\+Thing\+Smart\+SDK)}} start\+With\+App\+Key\+:@\char`\"{}\+YOUR\+\_\+\+APP\+\_\+\+KEY\char`\"{} secret\+Key\+:@\char`\"{}\+YOUR\+\_\+\+APP\+\_\+\+SECRET\char`\"{}\mbox{]};}.
