\doxysubsection{Detailed Description}
\mbox{\hyperlink{interface_thing_smart_speech_service}{Thing\+Smart\+Speech\+Service}} is used for voice control. 

There\textquotesingle{}s two main functions\+:
\begin{DoxyEnumerate}
\item Audio to text. Convert voice data into text strings.
\item Execute text command. Device related commands can be done through this API. 
\end{DoxyEnumerate}
