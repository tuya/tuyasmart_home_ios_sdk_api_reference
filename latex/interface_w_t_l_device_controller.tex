Set up a commissioning session for a device, using the provided setup payload to discover it and connect to it.


\begin{DoxyParams}{Parameters}

\hline
{\em new\+Node\+ID} & the planned node id for the node. @error error indication if discovery can\textquotesingle{}t start at all (e.\+g. because the setup payload is invalid).\\

\end{DoxyParams}
The IP and port for the device will be discovered automatically based on the provided discriminator.

Then a PASE session will be established with the device, unless an error occurs. WTLDevice\+Controller\+Delegate will be notified as follows\+:


\begin{DoxyItemize}
\item Discovery fails\+: on\+Status\+Update with WTLCommissioning\+Status\+Failed.
\item Discovery succeeds but commissioning session setup fails\+: on\+Pairing\+Complete with an error.
\item Commissioning session setup succeeds\+: on\+Pairing\+Complete with no error.
\end{DoxyItemize}
