Initialize the \mbox{\hyperlink{interface_w_t_l_o_t_a_header}{WTLOTAHeader}} with the given Matter OTA software image data (as defined in the \char`\"{}\+Over-\/the-\/\+Air (\+OTA) Software Update File Format\char`\"{} section of the Matter specification). The provided data is expected to point to a large enough initial chunk of an OTA software image that it includes the entire header (e.\+g. the entire image).

If the passed-\/in data is too small and does not contain the entire OTA image header, init\+With\+Data will return nil and the caller should try creating a new \mbox{\hyperlink{interface_w_t_l_o_t_a_header}{WTLOTAHeader}} object and initializing it with a larger chunk of the image.
