\doxysubsection{Detailed Description}
\mbox{\hyperlink{interface_w_t_l_subscribe_params}{WTLSubscribe\+Params}} This is used to control the behavior of attribute/event subscribes. If not provided (i.\+e. nil passed for the \mbox{\hyperlink{interface_w_t_l_subscribe_params}{WTLSubscribe\+Params}} argument), will be treated as if a default-\/initialized object was passed in.

Whether the subscription should automatically try to re-\/establish if it drops. The default value is YES.

If NO, loss of subscription will simply lead to an error report. Some subscription APIs do not support this value.
