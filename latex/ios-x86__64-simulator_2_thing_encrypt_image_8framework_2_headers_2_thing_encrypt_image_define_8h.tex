The options to control image operation. 

Load the image from remote and refresh the image cache.

Do not load image from/to disk cache.

Do not change the view\textquotesingle{}s image before set a new URL to it.

By default, placeholder images are loaded while the image is loading. This flag will delay the loading of the placeholder image until after the image has finished loading.

Use NSURLCache instead of Thing\+Image\+Cache.

Handles cookies stored in NSHTTPCookie\+Store.
