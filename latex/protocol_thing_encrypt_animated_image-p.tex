\doxysubsection{Detailed Description}
The \mbox{\hyperlink{protocol_thing_encrypt_animated_image-p}{Thing\+Encrypt\+Animated\+Image}} protocol declares the required methods for animated image display with Thing\+Encrypt\+Animated\+Image\+View.

Subclass a UIImage and implement this protocol, so that instances of that class can be set to Thing\+Encrypt\+Animated\+Image\+View.\+image or Thing\+Encrypt\+Animated\+Image\+View.\+highlighted\+Image to display animation.
