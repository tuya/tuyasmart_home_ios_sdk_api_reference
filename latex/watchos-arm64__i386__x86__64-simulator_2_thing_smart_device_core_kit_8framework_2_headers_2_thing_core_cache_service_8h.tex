Adds a device to the cache.

After you add the device to the cache, subscribe to the topic of the device\textquotesingle{}s MQTT messages. Make sure that the local key and protocol version are both configured. After the device is added, you can accept callbacks by implementing {\ttfamily device\+Did\+Add\+:} of {\ttfamily \mbox{\hyperlink{protocol_thing_core_cache_service_delegate-p}{Thing\+Core\+Cache\+Service\+Delegate}}}. The tasks to add devices, subscribe to messages, and execute callbacks are run in the asynchronous thread. The asynchronous queue.
